# JinkFi Documentation

![](.gitbook/assets/hero-banner.png)

Welcome to the official JinkFi documentation. JinkFi is a full-stack decentralised finance (DeFi) protocol built natively on **Tempo** (chainId 4217). It combines every tool a DeFi user needs into a single, permissionless interface.

---

## What can you do on JinkFi?

| Feature | Description |
|---|---|
| 🔄 **Swap** | Best-price token swaps via Uniswap V2 & V3 routing |
| 💧 **Liquidity Pools** | Provide V2 or concentrated V3 liquidity and earn fees |
| 🌾 **Farming** | Deposit LP tokens to earn JINK reward emissions |
| 💰 **Staking** | Stake tokens in audited third-party pools |
| 🔒 **Locker** | Time-lock tokens or LP positions on-chain |
| 🏆 **Quests** | Complete tasks to earn XP and protocol rewards |
| 🌉 **Bridge** | Move assets across chains via Li.Fi aggregation |
| 📈 **Perpetuals** | Trade perpetual futures up to 100× leverage *(coming soon)* |

## Protocol at a Glance

- **Native Chain:** Tempo Mainnet (chainId 4217)
- **DEX Engine:** Uniswap V2 + V3 (deployed natively on Tempo)
- **Authentication:** Sign-In With Ethereum (SIWE)
- **Token Standard:** ERC-20
- **Locker:** TokenLockerManagerV1 — non-custodial, on-chain time-lock
- **Farm:** JinkFarm — single-reward emission farm
- **Quests:** On-chain + social task verification
