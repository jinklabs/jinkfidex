# Quests

![](<../.gitbook/assets/quests-overview.png>)

The Quests page is JinkFi's engagement and rewards system. Projects post on-chain or off-chain tasks — users complete them to earn XP points that feed into the leaderboard and future reward distributions.

---

## How Quests Work

```
Project creates quest → defines tasks + XP rewards
           ↓
Users connect wallet → browse & filter quests
           ↓
Complete each task → XP is recorded on-chain
           ↓
Leaderboard ranks wallets by total XP earned
```

---

## Quest Types

| Quest Type | Description |
|---|---|
| **Featured** | Highlighted quests with higher rewards, curated by JinkFi |
| **Active** | Currently open quests within their start/end window |
| **All** | Every quest including upcoming and expired |

---

## Finding Quests

![](<../.gitbook/assets/quests-list.png>)

Use the filter tabs to narrow quests by status:

- **ALL** — shows every quest
- **ACTIVE** — only quests currently accepting submissions
- **FEATURED** — hand-picked high-value quests

Each quest card shows:

| Field | Description |
|---|---|
| **Title** | Quest name and description |
| **Tasks** | Number of tasks to complete |
| **XP** | Total points available |
| **End Date** | Deadline to complete |

---

## Completing Tasks

![](<../.gitbook/assets/quests-detail.png>)

1. Click a quest card to open the detail page
2. Review each task listed under the quest
3. Complete the task (e.g., make a swap, add liquidity, follow on X)
4. Click **Verify** — JinkFi checks on-chain or off-chain proof
5. XP is awarded to your wallet automatically

{% hint style="info" %}
On-chain tasks (swaps, LP deposits) are verified automatically. Social tasks may require a short review period.
{% endhint %}

---

## XP and Leaderboard

All XP earned across quests accumulates on your wallet address. Visit the [Leaderboard](../features/leaderboard.md) to see your rank against other participants.

---

## Creating a Quest (Projects)

Projects can submit quests for their community:

1. Click **Create Quest** on the Quests page
2. Define:
   - Quest title and description
   - Start and end dates
   - One or more tasks with XP values
3. Pay the listing fee in the native currency (USD on Tempo)
4. The JinkFi team reviews and publishes within 48 hours

![](<../.gitbook/assets/quests-create.png>)

{% hint style="warning" %}
The listing fee is non-refundable. Ensure all task descriptions are accurate before submitting.
{% endhint %}
