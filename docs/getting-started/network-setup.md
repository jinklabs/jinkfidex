# Set Up Tempo Network

JinkFi is deployed on **Tempo Mainnet**. The app will prompt you to add the network automatically, but you can also add it manually.

---

## Tempo Network Details

| Field | Value |
|---|---|
| **Network Name** | Tempo |
| **Chain ID** | 4217 |
| **RPC URL** | `https://rpc.tempo.xyz` |
| **Currency Symbol** | USD |
| **Currency Decimals** | 18 |
| **Block Explorer** | `https://explore.mainnet.tempo.xyz` |

---

## Add Manually in MetaMask

1. Open MetaMask → click the network selector at the top
2. Click **"Add Network"** → **"Add a network manually"**
3. Fill in the details from the table above
4. Click **Save**

## Native Currency

{% hint style="info" %}
Tempo's native currency is **USD** (18 decimals). You pay gas fees in USD — there is no volatile native token required. This makes fee estimation straightforward: 1 USD on Tempo = $1.
{% endhint %}

---

## Getting USD for Gas

To transact on Tempo, you need a small amount of USD in your wallet for gas. You can:

1. **Bridge** USD or stablecoins from Ethereum / Base using the [Bridge](../features/bridge.md) feature
2. Use the Tempo faucet (testnet) if you are testing
3. Ask in the [Discord](https://discord.gg/jinkfi) community
